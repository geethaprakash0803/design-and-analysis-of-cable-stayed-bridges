# Design and Analysis of Cable-Stayed Bridges

This repository contains files and documents related to the **design and structural analysis of cable-stayed bridges**, including models, reports, and presentations.

## Project Structure

* `models/` – Structural analysis models and design files
* `analysis/` – Calculation files and analysis results
* `presentations/` – Project presentation slides
* `reports/` – Final reports and documentation

## About

Cable-stayed bridges are a type of long-span bridge where the deck is directly supported by cables connected to towers. This project includes modelling, analysis, and design results for such a bridge using appropriate engineering tools and methods. ([IJSREM][1])

## Tools & Software

* STAAD.Pro (or similar structural analysis software)
* Microsoft PowerPoint
* PDF viewers/readers

## Notes

* The repository contains **final deliverables** such as design reports, models, and presentations.
* Intermediate or work-in-progress files should be kept separate if not part of the final output.
